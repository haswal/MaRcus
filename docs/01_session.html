Homework Exercises

  1. Create an excel file (in Microsoft Excel) with three columns of data. The first column should include both words and numbers and be 20 rows long. Read you file into R using the read_xlsx() function. Try three different ways to specify the sheet you want imported. Add missing values at random to the data, using -99 to indicate missing values. Read in the data again, and make sure missing values are handled correctly.

  2. Assign the “correct” colors to the colors variable in the plot above. Take a look at this link (Example 2) to figure out how to change the default color scheme.

    What did this example teach you about where the aes() command can be placed?

    Run ?scale_color_manual in the R console (this will only work if the ggplot2 package has been loaded). What happens? Find the paper talking about color vision deficiencies (and read it if you are interested).

    What does the c() function do?

  3. Make all dots in the plot squares. Use the text book or the internet to figure out how it is done. What did you learn about shapes and colors?

  4. A colleague of yours, from one of the Scandinavian countries (you don’t remember which, it could be any of them) has sent you a csv file. The data can be downloaded here. Read in the data using one of the functions from the readr package and make sure all columns are correctly read. You will have to open the data and figure out a way to identify where the data comes from (and use google to find the ICU locale language code). Then complete this code:

read_csv(file = ___,
         na = ___,
         locale = ___,
         col_types = cols(
           values = ___,
           date = ____
         ))

More information can be found in the R4DS book, chapters 11.3.4 and 11.4.2. Make sure you include everything necessary to avoid all warning messages.
